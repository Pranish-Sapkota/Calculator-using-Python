A **simple command-line calculator** in Python that performs basic arithmetic operations, keeps a **history of calculations**, and allows you to **manage your history**. Perfect for quick calculations right in your terminal! ✨

---

## ⚡ Features

- **Basic Arithmetic:** Addition (`+`), Subtraction (`-`), Multiplication (`*`), Division (`/`)  
- **History Management:**  
  - `history` → View past calculations in **reverse order**  
  - `clear` → Clear all saved calculations  
- **Error Handling:**  
  - Invalid input format detection  
  - Division by zero prevention  
- **Persistent Storage:** Calculations are saved in `history.txt` automatically

---

## 🛠️ Requirements

- Python 3.x

---

## 🚀 How to Use

1. **Run the program:**  

   ```bash
   python calculator.py

## 2. Enter calculations in the format:

    number operator number like (2 + 2)

## 3. Use commands:

- `history` → Show past calculations  
- `clear` → Clear all saved calculations  
- `exit` → Quit the program
